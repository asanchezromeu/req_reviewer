INCOSE Requirements Writing Principles Summary:
Clarity: Requirements should be clear and unambiguous.
Correctness: Requirements should accurately describe the need.
Completeness: Requirements should include all necessary information.
Feasibility: Requirements should be achievable within constraints.
Necessity: Each requirement should be necessary and relevant.
Conciseness: Requirements should be brief and to the point.
Consistency: Requirements should not conflict with each other.
Singularity: Each requirement should express a single idea.
Verifiability: Requirements should be testable.
Traceability: Requirements should be traceable to their source.

Example Requirement for Review just intended for context reference:
System Requirement example:"The camera system should capture high-resolution images in all lighting conditions."

Output format Feedback example:
Clarity: The term "high-resolution" needs to be defined more precisely. Suggest specifying the resolution in megapixels or another standard measure.
Correctness: Ensure the requirement accurately reflects the need without ambiguity.
Completeness: The requirement does not specify what constitutes "all lighting conditions." Define the range of lighting conditions.
Feasibility: Consider if it is technically possible to achieve high-resolution images in extreme lighting conditions. Suggest including technical constraints.
Necessity: Confirm if capturing high-resolution images in all lighting conditions is essential for the project's success.
Conciseness: The requirement is relatively concise, but specifying terms could help.
Consistency: Ensure this requirement does not conflict with other requirements.
Singularity: The requirement focuses on a single aspect of the camera system's performance.
Verifiability: Define how the performance in all lighting conditions will be tested.
Traceability: Ensure this requirement can be traced back to a user need or system objective.
Suggested Improved Requirement: "The camera system shall capture images at a resolution of at least 12 megapixels in lighting conditions ranging from 0.1 lux to 100,000 lux."
